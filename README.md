# Illustration Sharing Platform

A full-stack illustration sharing website inspired by Pixiv / DeviantArt.
Users can register, login, upload artworks, browse posts, and manage their own content.

## Tech Stack
Frontend: React, Axios  
Backend: Node.js, Express  
Database: MySQL  
Auth: Session-based authentication  
Hosting: Vercel (frontend), Render (backend)

## Features
- User authentication (login/logout)
- Upload artworks with images
- Browse artworks feed
- View post details
- Delete own posts
- Session-based login persistence
- Pagination & search
